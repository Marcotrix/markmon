# MARK MONitor
### Discord.py bot that tracks system resources
## Installation
1. Clone the repository
```git clone https://github.com/Marcotrix/markmon```

2. Create a .env file
```
TOKEN=YOUR_TOKEN_HERE
```
3. Install the required modules
```
pip install discord psutil dotenv
```
4. Run the bot
```
python3 main.py
```
